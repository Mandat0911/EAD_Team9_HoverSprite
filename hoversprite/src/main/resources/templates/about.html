Back-End

Front-End

During Front-End, using external stylesheets such as bootstrap.min.css and bootstrap-icons.css gave the team extra opportunities to express consistency across the website. Additionally, key-frame animations in CSS were added to enhance the user experience by animating images.

MySQL

With MySQL, each user has a stored password and specific roles for logging in. Behind the scenes, each group member has their own root password for MySQL, and a shared schema allows each member full access to the website's database.
